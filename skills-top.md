# Top Page UI Skills

## Goal

- Maximize appeal as a landing page
- Capture attention immediately in the first view

---

## First View

- Strong visuals are required
- Make the catchcopy stand out
- Place a CTA

---

## Flow

- Structure the page as a story:
  - Problem → Solution → Track Record → CTA

---

## Visual

- Use rich decorative elements
- Strengthen motion and visual accents

---

## CTA

- Place multiple CTAs throughout the page
- Use a visually prominent design
